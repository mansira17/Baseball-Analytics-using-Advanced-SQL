# Baseball-Analytics-using-Advanced-SQL-(MySQL)

## Project Overview

This project performs an in-depth analysis of baseball data using **advanced SQL techniques**.  
The analysis spans **school pipelines, team salary trends, player career trajectories, and player comparisons**, leveraging:

- Common Table Expressions (CTEs)
- Window functions
- Joins and aggregations
- Date and time functions

The project focuses on extracting **real-world analytical insights** relevant to **sports analytics, financial trends, and player performance evaluation**, using **MySQL**.

## Tools & Technologies

### Database
- MySQL

### SQL Concepts Used
- **Common Table Expressions (CTEs)**
- **Window Functions**
  - `ROW_NUMBER()`
  - `NTILE()`
  - `LAG()`
  - `SUM() OVER`
- **Aggregate Functions**
- **Date & Time Functions**
- **Joins & Subqueries**


## Analysis Areas

- School-to-player pipeline analysis by decade
- Team salary trends and cumulative spending
- Identification of top-spending teams
- Player career length and age-based analysis
- Player comparisons (birthdays, batting preferences)
- Decade-over-decade changes in player physical attributes

## Highlights

- Identified **top schools producing MLB players by decade**
- Calculated **team-wise cumulative salary spend**, including first year exceeding \$1B
- Analyzed **player career longevity** and debut-to-retirement trends
- Compared **batting preferences** across teams
- Measured **decade-over-decade changes** in player height and weight
